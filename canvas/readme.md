 2. Objective

________________________________________________________________________

 3. Project Description

- Luxury Mode is an interactive frontend webpage
- It displays animated sparkles and floating bubbles
- It uses glassmorphism effect for modern UI design
- It contains interactive cards with hover effects
- JavaScript is used to create mouse-movement interaction
- The canvas moves dynamically based on cursor position

 7. Working of the Project

- Sparkles animation runs continuously in background
- Bubbles rise from bottom using CSS animation
- Canvas is displayed at center of screen
- Cards animate when hovered
- JavaScript tracks mouse movement
- Canvas moves according to mouse position
- This creates interactive visual effect
