 3. Project Description

- Cafe Charm is a frontend coffee shop website
- It displays information about the cafe
- Users can view menu items and prices
- It includes Navbar, Hero Section, About Section, Menu Section, and Footer
- It provides a modern and user-friendly interface
- The website is fully responsive
